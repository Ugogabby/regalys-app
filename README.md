\# REgalys

\*\*Real-world Evidence Generation and Analysis Insights\*\*



REgalys is a domain-specific RAG system for pharmacoepidemiology methods and real-world evidence generation, covering causal inference study design, time-to-event methodology, confounder control, and administrative claims data methods across therapeutic areas.



Live app: https://regalys.streamlit.app
